# Panduan Update GitHub

Setiap kali Anda selesai melakukan perubahan pada kode (edit, tambah, atau hapus file), ikuti langkah-langkah berikut untuk meng-update repository GitHub Anda:

## 1. Cek Status (Opsional)
Untuk melihat file apa saja yang berubah:
```bash
git status
```

## 2. Tambahkan Perubahan
Untuk menyiapkan semua perubahan agar siap disimpan:
```bash
git add .
```

## 3. Simpan Perubahan (Commit)
Simpan perubahan dengan pesan yang menjelaskan apa yang Anda lakukan:
```bash
git commit -m "Tulis pesan perubahan Anda di sini"
```
*Contoh: `git commit -m "Menambahkan fitur login"`*

## 4. Upload ke GitHub (Push)
Kirim perubahan ke GitHub:
```bash
git push
```

---

**Tips:**
- Lakukan ini secara berkala agar progress Anda tersimpan aman di GitHub.
- Jika ada pesan error, baca pesannya baik-baik atau tanyakan pada asisten AI.
